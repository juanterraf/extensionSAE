Procesando...

Como usar

  • Abri consultaexpedientes.justucuman.gov.ar en otra pestaña
  • Navega a cualquier expediente y la extension lo detecta automaticamente
  • Tambien podes buscar desde la pestaña Buscar seleccionando centro y fuero

Seguimiento

  • Usa "Seguir expediente" para monitorear cambios
  • La extension verifica automaticamente cada 30 minutos
  • Si hay movimientos nuevos, recibis una notificacion de Chrome
  • Los expedientes con novedades se resaltan en la pestaña Seguimiento

Importacion masiva

  • Prepara un Excel (.xlsx) o CSV con una columna de numeros de expediente
  • Formato esperado: 1234/23 o 1234-2023
  • Selecciona centro judicial y fuero para la busqueda
  • La extension buscara en todos los fueros si no encuentra en el seleccionado
  • Necesitas tener la pagina del SAE abierta en el buscador (para el captcha)

Descarga

  • Desde un expediente, usa "Descargar todo (ZIP)" para obtener PDFs y textos
  • Cada tramite se descarga como PDF si esta disponible, o como texto
  • Se incluye un informe general automatico en el ZIP

Informe con IA

  • Obtene una API key gratis en aistudio.google.com
  • Pegala en la configuracion de la pestaña Expediente
  • Desde un expediente, usa "Informe IA" para obtener un analisis inteligente
